# Руководство по тестированию MVP

## Запуск приложения

```bash
npm run dev
```

Откройте браузер по адресу `http://localhost:5174` (или другому порту, который укажет Vite).

## Чек-лист тестирования

### 1. Базовая функциональность карты

- [ ] Карта загружается и отображается на весь экран
- [ ] Тайлы OpenStreetMap загружаются корректно
- [ ] Атрибуция "© OpenStreetMap contributors" видна в правом верхнем углу

### 2. Управление картой

#### Desktop
- [ ] Zoom колесом мыши работает
- [ ] Кнопки +/- для zoom работают
- [ ] Карта перетаскивается мышью (pan)
- [ ] Карта ограничена территорией России (не улетает далеко за границы)

#### Mobile/Touch
- [ ] Pinch-zoom двумя пальцами работает
- [ ] Карта перетаскивается одним пальцем
- [ ] Кнопки +/- доступны и работают

### 3. Маркеры и кластеры

- [ ] При начальном zoom видны кластеры (зеленые круги с цифрами)
- [ ] При приближении кластеры распадаются на отдельные маркеры
- [ ] При наведении на маркер появляется tooltip с названием деревни
- [ ] При отдалении маркеры снова собираются в кластеры

### 4. Взаимодействие с кластером

- [ ] Клик по кластеру открывает панель со списком деревень
- [ ] Панель показывает количество деревень в кластере
- [ ] Список отсортирован по алфавиту (русская локаль)
- [ ] В каждой строке: название, регион, количество треков
- [ ] Клик по кластеру НЕ зумит карту (важно!)
- [ ] Карта плавно перемещается к центру кластера

### 5. Взаимодействие с маркером

- [ ] Клик по маркеру открывает карточку деревни
- [ ] Карта плавно перемещается к маркеру (flyTo animation)
- [ ] Карточка показывает: название, регион, список треков
- [ ] Если треков нет - показывается соответствующее сообщение
- [ ] Ссылка "Открыть треки" работает и открывается в новой вкладке

### 6. Список деревень в кластере

- [ ] Клик по деревне в списке закрывает список
- [ ] Клик по деревне в списке открывает карточку деревни
- [ ] Карта зумится и перемещается к выбранной деревне
- [ ] Кнопка "Закрыть" закрывает панель

### 7. Карточка деревни

- [ ] Треки отображаются списком (title и artist)
- [ ] Кнопка "Закрыть" закрывает карточку
- [ ] Ссылка на скачивание имеет атрибуты `target="_blank" rel="noopener noreferrer"`

### 8. Клик по пустой карте

- [ ] Клик по пустому месту карты закрывает все открытые панели

### 9. Адаптивность UI

#### Desktop (ширина > 768px)
- [ ] Панель отображается справа как боковая панель
- [ ] Панель имеет фиксированную ширину ~380px
- [ ] Панель имеет border и border-radius
- [ ] Панель занимает высоту от 12px сверху до 12px снизу

#### Mobile (ширина < 768px)
- [ ] Панель отображается снизу как bottom sheet
- [ ] Панель имеет max-height: 60vh
- [ ] Панель имеет скругленные верхние углы
- [ ] Панель не перекрывает атрибуцию

### 10. Transitions и анимации

- [ ] Панели появляются/исчезают с плавной анимацией
- [ ] При `prefers-reduced-motion` анимации отключаются
- [ ] Переход к деревне (flyTo) плавный и занимает ~0.35s

### 11. Обработка ошибок

#### Тест с отключенной сетью
- [ ] При ошибке загрузки JSON появляется панель с ошибкой
- [ ] Панель содержит текст ошибки
- [ ] Кнопка "Повторить" работает
- [ ] При успешной повторной загрузке панель закрывается

#### Тест с некорректными данными
Временно измените `villages.json`:
- [ ] Приложение не падает при некорректных координатах
- [ ] Приложение не падает при отсутствующих полях
- [ ] Некорректные записи фильтруются и не отображаются

### 12. Производительность

- [ ] Карта не фризит при взаимодействии
- [ ] Scroll/zoom плавные без лагов
- [ ] Переключение между панелями мгновенное
- [ ] Нет заметных задержек при клике на маркеры/кластеры

### 13. Проверка в разных браузерах

- [ ] Chrome/Edge (Chromium)
- [ ] Firefox
- [ ] Safari (если на macOS)
- [ ] Mobile Safari (iOS)
- [ ] Chrome Mobile (Android)

### 14. Данные

Проверьте конкретные деревни:
- [ ] Архангельская область (1 трек)
- [ ] Донские казаки (5 треков)
- [ ] Некрасовцы (7 треков)
- [ ] Все координаты корректны (точки в России)
- [ ] Ссылки на скачивание открываются

## Известные ограничения MVP

1. ❌ Нет поиска по названию
2. ❌ Нет фильтров по регионам
3. ❌ Нет встроенного аудиоплеера
4. ❌ Данные статичные (без backend)
5. ❌ Ссылки временные (example.com)

## Отладка

### Открыть консоль разработчика

**Chrome/Firefox/Edge:**
- Windows/Linux: `F12` или `Ctrl+Shift+I`
- macOS: `Cmd+Option+I`

### Проверить ошибки

В консоли не должно быть:
- ❌ JavaScript errors
- ❌ Failed network requests (кроме favicon)
- ❌ Vue warnings

### Проверить сеть (Network tab)

- ✅ `/data/villages.json` - статус 200
- ✅ Leaflet CSS/JS загружены
- ✅ Тайлы OpenStreetMap загружаются (много запросов к `tile.openstreetmap.org`)

## Проблемы и решения

### Маркеры не видны (только квадраты)

**Проблема:** PNG иконки не загружаются  
**Решение:** Проверьте фикс в `src/main.ts` (импорт PNG через Vite)

### Карта серая/пустая

**Проблема:** Контейнер не имеет высоты  
**Решение:** Проверьте CSS (`html, body, #app { height: 100% }`)

### Панели не видны

**Проблема:** Tailwind не подключен или не скомпилирован  
**Решение:** Проверьте `src/index.css` и `tailwind.config.js`

### JSON не загружается

**Проблема:** 404 ошибка  
**Решение:** Убедитесь что файл в `public/data/villages.json`
